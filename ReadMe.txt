This project contains all code used to develop and run Quest For Information. 

Quest-for-Information-Demo 
- the folder contains the implementation of the Unity Game Client and the Python-based LLM Server. 

LLM-Comparison
- the folder contains reusable JupyterLab notebooks for running local inference and testing of existing LLMs.

LLM-Customization
- the folder contains reusable JupyterLab notebooks showcasing different methods of adapting existing LLMs to a game character
- additional notebooks contain approaches to testing these methods
